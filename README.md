This project uses Python and Pandas to analyze the historical performance of S.E. Palmeiras over the last 6 seasons. As a Computer Science student and soccer enthusiast, I developed this tool to visualize offensive trends and win-rate consistency.

Technologies Used:

  Python 3.x

  Pandas: For data manipulation and cleaning.

  Matplotlib & Seaborn: For data visualization and trend analysis.

Key Features:

  Data processing of seasonal statistics.

   Calculation of custom metrics like Win Rate percentage.

  Visual trend analysis of goals scored per year.

How to Run:

   Clone this repository.

  Install dependencies: pip install pandas matplotlib seaborn.

  Run analysis.py.
